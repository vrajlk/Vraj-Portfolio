My Portfolio Some Of My Works

Verseline

Verseline is a music streaming web app built using JavaScript, Node.js, and HTML/CSS. It dynamically fetches songs from a local server, displaying them in a responsive, card-based layout. The app features an audio player with play/pause, skip, and volume control, as well as a seekbar and time conversion for enhanced user experience. The song data, including album metadata, is fetched in real-time through Node.js, and the left menu is responsive, collapsing for screen sizes below 1400px. This project strengthened my skills in full-stack JavaScript development and client-server architecture, while ensuring mobile accessibility.

Random Jokes Generator

This project is an interactive web application designed to deliver a variety of jokes with a focus on dark and programming humor. Built using HTML, CSS, and JavaScript, it fetches jokes from a reliable joke API and dynamically parses the JSON data to display the text in a user-friendly format. The sleek, modern layout incorporates a 3D button effect for enhanced user interaction. The app provides a seamless experience by offering refreshing humor with every click, making it an ideal tool for quick entertainment and a touch of tech humor.

Smart Parking System

This innovative parking management system combines ESP8266(NODE MCU), C/C++, HTML, CSS, JavaScript, PHP, and MySQL to offer a seamless, efficient solution. Key features include: Real-Time Slot Monitoring: Dynamic LED display updates parking space availability. Automated Payment & Ticketing: Streamlined entry, exit, and payment process. User-Friendly Web Interface: Convenient remote checking of parking slots. Impact: Transforms parking with faster slot identification, reduced manual work, and optimized space usage, boosting user satisfaction and operational efficiency.
